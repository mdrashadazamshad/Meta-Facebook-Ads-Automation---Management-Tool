# Meta/Facebook Ads Automation Tool

A Node.js tool to manage and analyze Facebook/Meta Ads campaigns. Designed to optimize ad performance and track key metrics.

## Features
- Connect to Meta Ads API
- Track ad performance
- Budget & ROI monitoring
- Customizable reporting

## Usage
1. Add access token in `config/fb-config.json`
2. Run `npm install`
3. Run `npm start`
